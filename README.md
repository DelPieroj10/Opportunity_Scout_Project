# Opportunity Scout

**Opportunity Scout** is a research workspace designed to help teams and founders discover market opportunities faster.

Instead of spending days manually reviewing competitor sites, customer reviews, pricing pages, and discussion forums, the system coordinates multiple research streams simultaneously. It delivers a structured summary of opportunities, risks, and recommendations for a given market category.

> 🚧 Project currently under active development, created as a technical exercise and portfolio piece.

## ✨ What does it analyze?

- 🏢 Competitors
- 💰 Pricing strategies
- 🗣️ Customer reviews
- 📍 Brand positioning
- 📈 Market trends
- ⚠️ Frequently reported issues

### Note on shadcn/ui + Base UI

This project uses the shadcn/ui variant built on **Base UI** instead of Radix UI. The main difference to keep in mind when adding or modifying components:

- Radix UI uses the `asChild` (boolean) + `children` pattern.
- Base UI uses the `render` pattern (receives the full element as the prop value).

```tsx
// Base UI pattern used in this project
<SidebarMenuButton
  render={
    <a href="/ruta">
      <span>Texto</span>
    </a>
  }
/>
```

## 🚀 How to run the project

### Prerequisites

- Node.js 18+
- [pnpm](https://pnpm.io/installation)

### Installation

```bash
pnpm install
```

### Development mode

```bash
pnpm dev
```

This starts the Vite server (default: `http://localhost:5173`).

### Production build

```bash
pnpm build
```

### Add a new shadcn component

```bash
pnpm dlx shadcn@latest add <component-name>
```
